# XML-XSD-XSLT-RELEVE-BANCAIRE
L'objectif de ce fichier est de stocker les informations d'un compte bancaire de manière structurée et lisible par une machine. Il sépare clairement les données (solde, montants) de leur présentation, facilitant ainsi l'échange de ces informations entre différents systèmes informatiques (par exemple, entre une banque et un logiciel de comptabilité)
